My Bash Profile

Being on a remote server without your bash profile sucks. Custom keyboard shortcuts don't work, you don't know how to use vim without your plugins, and you can't remember how exactly to search the man pages. Be lost no more. With My Bash Profile you can simply curl your bash profile onto the server and be off.
